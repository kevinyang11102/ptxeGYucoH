# 停车场管理系统 java366

## 项目概述


## 技术栈

- **开发工具**：IDEA 2020.3 以上，VSCode
- **运行环境**：
  - JDK 1.8
  - Maven 3.6.0
  - MySQL 5.7
  - Node.js 14
- **后端技术**：Springboot + Mybatis-Plus
- **前端技术**：HTML + CSS + Layui + jQuery + Bootstrap + Vue + Axios + Element-UI

## 功能模块

### 用户角色

- 个人信息管理
- 预定停车位
- 缴费信息管理
- 下载近五天缴费记录

### 管理员角色

- 用户信息管理
- 车辆信息管理
- 车辆费用设置
- 停泊车辆管理
- 车辆进出管理
- 登录日志查询

## 系统角色

- 用户：普通用户权限，可进行个人操作
- 管理员：高级权限，可管理用户信息及系统设置

## 目录结构

```
|-- 前端代码

|       |-- components

|-- 后端代码
|   |-- src
|       |-- main
|       |-- resources
|       |-- sql
|-- 其他文档
```

## 部署步骤

1. 环境准备：确保JDK、Maven、MySQL和Node.js等环境正常安装。
2. 数据库构建：根据提供的SQL脚本构建数据库。
3. 后端部署：使用IDEA或其他工具编译后端代码，并运行主程序。
4. 前端部署：在项目前端目录下，执行`npm install`安装依赖，然后执行`npm run dev`启动前端服务。

## 核心亮点

- 前后端分离，易于维护和扩展。
- 基于成熟的Springboot和Vue技术栈，保证了系统的稳定性与高效性。
- 系统角色清晰，权限管理明确。
- 提供完整的软件环境安装包，便于快速部署和使用。
